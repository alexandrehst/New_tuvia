Tela: principal_plano.md

um plano estratégico é:
plano
|
-- Objetivo
|   |
|    -- Resultado-chave
|
-- Plano de um departamento ou segundo nível
   |
    -- objetivo
       |
       -- resultado-chave

Esse tela é responsável pelo gerencimaento do plano.
- Criar novos planos por departamento (okrs de nivel inferior)
- criar novos objetivos manualmente
- sugerir um novo objetivo (e kr) com a IA
- excluir qualquer um dos elementos
- mostra a lista de responsáveis por algum dos objetivos
- a edição do plano é em painel_editar_plano.png
- a edição do objetivo é com o painel painel_objetivo.png. Observe que os objetivos podem ter responsáveis atribuídos.
- a edição do resultado chave está em Painel_resultado_chave.png
- após a criação, o plano é publicado e passa para a fazer de acompanhamento, isto é, um clique no plano leva para acompanhamento_objetivos.md
